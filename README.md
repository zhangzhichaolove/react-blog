# React-Blog 使用说明

## 快速开始

> 克隆项目源码

```
git clone https://github.com/zhangzhichaolove/react-blog.git
```

> 安装项目依赖

```
cd react-blog
npm install
```

> 运行项目

```
npm start
```

> 浏览器访问[http://localhost:3000](http://localhost:3000)
